## 🧭 Overview

marketing-ai-studio is an AI-native marketing app for Windows. It helps you create brand assets, SEO content, social posts, and video content from one place.

It uses a multi-agent system to handle different marketing tasks. Each agent focuses on one job, such as writing copy, checking search terms, or shaping visuals. The app uses FastAPI for the backend, React JS for the interface, Google ADK for agent control, and Gemini 3.1 and Veo 3 for content generation.

## ✨ What you can do

- Build brand assets from a simple prompt
- Create SEO-focused content for web pages and blogs
- Write social media posts for different platforms
- Generate content ideas for campaigns
- Create high-fidelity images and video content
- Run many marketing tasks from one workspace
- Let agents handle each step of the workflow

## 🧠 How it works

The app uses several agents, and each one handles one part of the task:

- A planning agent breaks the request into steps
- A writing agent creates copy
- An SEO agent checks keywords and search intent
- A branding agent shapes tone and style
- A media agent helps with image and video work
- A review agent checks the final output

This setup helps the app work like a small marketing team. You give one request, and the system routes the work to the right agents.

## 🛠️ Basic setup tips

If the app asks for a browser, use your default browser first. If that does not work, try Chrome.

If the app needs Google account access, make sure you are signed into the right account before you start.

If content generation feels slow, wait for each job to finish before starting a new one. Multi-agent workflows can take time because they run several steps in order.

If you want clean output, use one goal at a time. For example, ask for a blog outline first, then ask for the full draft.

## 🧰 Troubleshooting

### The file will not open
- Download the file again

- Try a shorter prompt
